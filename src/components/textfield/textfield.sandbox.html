Ne pas afficher un compteur sur un champ de saisie sans limite de caractères

Lorsque qu'on ajoute un compteur sur un champ sans limite de caractère, le compteur doit être masqué. Nous n'avons pas de cas d'utilisation de compteur de mots sans imposer de limite.

Test 3 : Retour à la ligne automatique (word-wrap)

Le texte qui "dépasse" du champ de saisi doit revenir à la ligne automatiquement lorsque la ropriété word-wrap est activée (plutôt que de cacher le texte en surplus).
From PR-260

Champ de saisie de type text et word-wrap à true

Le wordwrap est bien appliqué car le textfield est de type text.

Champ de saisie de type courriel et word-wrap à true

Le wordwrap n'est pas appliqué car le textfield n'est pas de type text. Envoie également une erreur dans la console.
